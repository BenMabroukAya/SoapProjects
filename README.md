# 🧼 SoapProjects

Ce dépôt regroupe une série de mini-projets et d'exemples illustrant l'utilisation des technologies SOAP et REST avec Java (JAX-WS, JAX-RS) ainsi que des notions de sérialisation/désérialisation XML.

## 📁 Structure du projet

### 1. Banque-JAX-RS
> Implémentation d'un service RESTful simulant des opérations bancaires simples (consultation, retrait, dépôt, etc.).

### 2. TP-OXM
> Exemples de sérialisation et désérialisation XML avec JAXB (Object-XML Mapping).

### 3. jax-rs-arc-client
> Client REST consommant un service "personne".

### 4. jax-rs-arc
> Serveur REST fournissant un service "personne".

### 5. jax-rs-client-php
> Client PHP consommant un service REST en Java.

### 6. jax-ws-client
> Client SOAP interagissant avec un service JAX-WS.

### 7. jax-ws-serveur
> Serveur SOAP exposant un web service avec JAX-WS (exemples simples avec sérialisation/désérialisation).

## 🛠️ Technologies utilisées

- Java (JDK 8+)
- JAX-RS (Jersey)
- JAX-WS
- JAXB
- PHP (pour le client REST)
- Apache Tomcat (ou autre serveur servlet)
- SOAP & REST API

## 🚀 Lancer un projet

Chaque sous-dossier contient un projet Maven prêt à être importé dans un IDE Java (comme IntelliJ IDEA ou Eclipse).  
Voici les étapes générales :

1. Cloner le repo :
   ```bash
   git clone https://github.com/BenMabroukAya/SoapProjects.git
   ```

2. Importer un des projets dans votre IDE.

3. Lancer l'application via une classe `main()` ou déployer sur un serveur Tomcat.
